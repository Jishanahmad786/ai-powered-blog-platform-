# 🤖 AI-Powered MERN Blogging Platform

A full-stack blogging web application built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.  
This platform allows admins to create blogs, generate content using **Gemini AI**, and manage user comments through an admin dashboard.

---

## 🚀 Features

- 📝 Create, edit, and manage blog posts
- 🤖 AI-generated blog content using **Google Gemini API**
- 🖼 Image upload & optimization using **ImageKit**
- 🔐 Secure **Admin Authentication (JWT)**
- 💬 Comment moderation system (approve / delete comments)
- ⚡ Fast and responsive UI using **React + Tailwind CSS**

---

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router
- Context API

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Multer (File Upload)

### Integrations
- Google Gemini AI
- ImageKit
